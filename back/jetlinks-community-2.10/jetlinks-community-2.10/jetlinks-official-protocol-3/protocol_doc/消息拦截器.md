本文档主要介绍当设备侧要求平台进行答复消息时如何自动回复和当平台下发指令时如何设置自动响应消息。

# 操作步骤
## 设备上报平台自动响应
由于平台无法自动响应消息，因此自动响应消息可以在协议包内处理报文时进行响应。

示例代码：

```java
@Nonnull
@Override
public Flux<DeviceMessage> decode(@Nonnull MessageDecodeContext context) {
	return ((FromDeviceMessageContext) context)
	.getSession()
    .send(EncodedMessage.simple(Unpooled.buffer().writeBytes("响应消息".getBytes())))	//此处响应消息作为示例，开发者自行更换为自有业务的回复内容
	.thenMany(doDecode(context.getMessage()));
}
```

## 平台下发指令自动回复
在某些场景下，由于设备在设计初期未设计响应回复，或者某些短链接设备无法响应平台指令，因此自动响应消息可以在协议包内封装报文时响应或者在平台发送消息后进行拦截处理响应（<font style="color:#DF2A3F;">此处的发送消息并非指通过网络发送消息，而是平台调用了发送方法后</font>）。

### 在协议包封装设备指令时自动回复
```java
@NonNull
@Override
public Publisher<? extends EncodedMessage> encode(@NonNull MessageEncodeContext context) {
	FunctionInvokeMessageReply deviceMessage = (FunctionInvokeMessageReply) context.getMessage();
	return context
	.reply(FunctionInvokeMessageReply.success(deviceMessage.getDeviceId(), deviceMessage.getFunctionId(), deviceMessage.getMessageId(), "默认回复"))
	.thenReturn(EncodedMessage.simple(
		wrapByteByf(
			BinaryMessageType.write(deviceMessage, Unpooled.buffer())
		)
	));
}
```

### 真实发送消息前的拦截器内自动回复
拦截器定义参考[消息拦截器](https://hanta.yuque.com/px7kg1/nn1gdr/ymfpop6hbuumen6o)

```java
package org.jetlinks.protocol.official;

import org.jetlinks.core.device.DeviceOperator;
import org.jetlinks.core.message.DeviceMessage;
import org.jetlinks.core.message.interceptor.DeviceMessageSenderInterceptor;
import reactor.core.publisher.Flux;
import reactor.core.publisher.Mono;

public class CustomMessageSenderInterceptor implements DeviceMessageSenderInterceptor {


    @Override
    public <R extends DeviceMessage> Flux<R> afterSent(DeviceOperator device, DeviceMessage message, Flux<R> reply) {
        if(message instanceof FunctionInvokeMessage){
			FunctionInvokeMessage functionInvokeMessage = (FunctionInvokeMessage) message;

			FunctionInvokeMessageReply functionInvokeMessageReply = new FunctionInvokeMessageReply();
			functionInvokeMessageReply.setFunctionId(functionInvokeMessage.getFunctionId());
			functionInvokeMessageReply.setDeviceId(functionInvokeMessage.getDeviceId());
			functionInvokeMessageReply.setMessageId(functionInvokeMessage.getMessageId());
			functionInvokeMessageReply.setOutput("自动回复功能调用......");
			return Flux.just(functionInvokeMessageReply)
					   .map(r -> (R) r);

		}

		return Flux.empty();
    }
}

```
