本文主要介绍 JetLinks 物联网平台协议包中，网关设备下查询子设备方法。

## <font style="color:rgb(51, 51, 51);">添加依赖</font>
在[官方协议](https://github.com/jetlinks/jetlinks-official-protocol)中的`pom.xml`添加 `device-manager` 依赖，用于查询设备实例信息，此处以2.3.0-SNAPSHOT为示例。

```xml
<dependency>
  <groupId>org.jetlinks.pro</groupId>
  <artifactId>device-manager</artifactId>
  <version>2.3.0-SNAPSHOT</version>
</dependency>
```

## <font style="color:rgb(51, 51, 51);">协议支持注册</font>
在 `ProtocolSupportProvider` 中，将 `LocalDeviceInstanceService` 注入到解码器。

```java
public class JetLinksProtocolSupportProvider implements ProtocolSupportProvider {


    @Override
    public Mono<CompositeProtocolSupport> create(ServiceContext context) {
        return Mono.defer(() -> {
            CompositeProtocolSupport support = new CompositeProtocolSupport();
            LocalDeviceInstanceService localDeviceInstanceService = context.getService(LocalDeviceInstanceService.class)
                                                                           .orElseThrow(IllegalStateException::new);
            //使用构造函数传入Bean
            JetLinksMqttDeviceMessageCodec mqttCodec = new JetLinksMqttDeviceMessageCodec(DefaultTransport.MQTT, localDeviceInstanceService);

            support.setId("jetlinks.v3.0");
            support.setName("JetLinks V3.0");
            support.setDescription("JetLinks Protocol Version 3.0");
            //MQTT
            support.addMessageCodecSupport(mqttCodec);
            return Mono.just(support);

        });

    }
}
```

## <font style="color:rgb(51, 51, 51);">消息解码器实现</font>
### <font style="color:rgb(51, 51, 51);">定义消息编码解码器</font>
```java
public class JetLinksMqttDeviceMessageCodec implements DeviceMessageCodec {

    private final Transport transport;

    private final ObjectMapper mapper;
    private LocalDeviceInstanceService service;


    public JetLinksMqttDeviceMessageCodec(Transport transport, LocalDeviceInstanceService localDeviceInstanceService) {
        this.transport = transport;
        this.mapper = ObjectMappers.JSON_MAPPER;
        this.service = localDeviceInstanceService;
    }


    @Nonnull
    @Override
    public Flux<DeviceMessage> decode(@Nonnull MessageDecodeContext context) {
        MqttMessage message = (MqttMessage) context.getMessage();
        byte[] payload = message.payloadAsBytes();

        return TopicMessageCodec
            .decode(mapper, TopicMessageCodec.removeProductPath(message.getTopic()), payload)
            .flatMap(msg -> {

                if (!(msg instanceof DeviceOnlineMessage)) {
                    return Flux.just(msg);
                }
                String deviceId = msg.getDeviceId();

                // 网关设备上线
                return Flux.concat(
                    Flux.just(msg),
                    queryChildrenOnline(deviceId)
                );
            })
            //如果不能直接解码，可能是其他设备功能
            .switchIfEmpty(FunctionalTopicHandlers
                               .handle(context.getDevice(),
                                       message.getTopic().split("/"),
                                       payload,
                                       mapper,
                                       reply -> doReply(context, reply)))
            ;

    }

}
```

### <font style="color:rgb(51, 51, 51);">子设备查询逻辑</font>
<font style="color:rgb(51, 51, 51);">通过 </font>`<font style="color:rgb(51, 51, 51);background-color:rgb(243, 244, 244);">LocalDeviceInstanceService</font>`<font style="color:rgb(51, 51, 51);">，根据 </font>`<font style="color:rgb(51, 51, 51);background-color:rgb(243, 244, 244);">parentId</font>`<font style="color:rgb(51, 51, 51);"> 查询该网关下的所有子设备。</font>

```java
private Flux<DeviceMessage> queryChildrenOnline(String gatewayId) {
     // 查询网关下的子设备
    QueryParamEntity query = new QueryParamEntity();
    query.noPaging();
    query.addTerm(Term.of("parentId", TermType.eq, gatewayId));

    return service
        .query(query)
        .map(device -> {
            // 子设备在线消息
            DeviceOnlineMessage online = new DeviceOnlineMessage();
            online.setDeviceId(device.getId());
            online.setTimestamp(System.currentTimeMillis());

            // 封装为子设备消息
            ChildDeviceMessage childDeviceMessage = new ChildDeviceMessage();
            childDeviceMessage.setDeviceId(gatewayId); // 网关ID
            childDeviceMessage.setChildDeviceMessage(online);

            return childDeviceMessage;
        });
}
```
