# 操作步骤
:::info
**<font style="color:#2F4BDA;">🌏</font>****<font style="color:#2F4BDA;"> 说明</font>**

此处以官方协议为示例。

:::

1. 在官方协议`JetLinksProtocolSupportProvider`中加入如下代码
    1. **<font style="color:#DF2A3F;">集群模式下的定时任务应使用</font>**`**<font style="color:#DF2A3F;">FluxCluster</font>**`**<font style="color:#DF2A3F;">类里面的方法注册定时任务调度。</font>**

```java
@Override
    public Mono<CompositeProtocolSupport> create(ServiceContext context) {
        return Mono.defer(() -> {
            CompositeProtocolSupport support = new CompositeProtocolSupport();

        	//{.....中间省略...}
        
            Disposable disposable = Flux
                .interval(Duration.ofSeconds(10))
                .subscribe(value -> {
                    log.info("定时任务开始，第{}次",value)
                    doTake();//根据场景业务执行相关方法
                 });

            support.doOnDispose(disposable);


            return Mono.just(support);
        });
    }
```

2. 打包并上传协议包。

:::info
**<font style="color:#2F4BDA;">🌏</font>****<font style="color:#2F4BDA;"> 说明</font>**

在项目启动或者上传协议包后，会加载协议包并执行该定时任务。

:::
